# 테스트 전략 및 케이스

## 1. 테스트 종류

- 컴포넌트 렌더링 테스트 (React Testing Library)
- 비즈니스 로직 유닛 테스트 (Jest)
- API/Mock 테스트

## 2. 예시 테스트 케이스

- StockList: 항목 렌더링, 삭제/수정 동작
- StockForm: 입력값 검증, 생성/수정 동작
- StockPieChart: 비중 계산 및 시각화
- 유틸리티 함수: 소수점, 수익률, 포맷팅 등

## 3. 테스트 실행 명령어

- `npm run test`
- `npm run typecheck`
- `npm run lint`
